# INSTAFOREX Search Engine

Разработка интерфейса поисковой системы для крупнейшего форекс-брокера. Проект включает проектирование сложной фильтрации, автокомплита и результатов поиска.

## Задачи проекта

- Проектирование интуитивного поискового интерфейса
- Разработка системы фильтров и сортировки
- Оптимизация скорости отображения результатов
- A/B тестирование различных вариантов UI

## Технологии

- Figma для дизайна
- Google Analytics для аналитики
- Jira для управления проектом

## Результаты

Сокращение времени поиска информации на 45%. Увеличение удовлетворенности пользователей на 30%.
