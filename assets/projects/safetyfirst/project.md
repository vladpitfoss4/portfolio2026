# SAFETY FIRST Web Application

Комплексное веб-приложение для управления безопасностью на производстве. Проект включает дизайн dashboard, систему уведомлений и мобильную версию.

## Задачи проекта

- Проектирование информационной архитектуры
- Разработка dashboard с real-time данными
- Создание системы уведомлений и алертов
- Адаптация под мобильные устройства

## Технологии

- Figma для дизайна
- Miro для user flow mapping
- After Effects для анимаций

## Результаты

Снижение времени реагирования на инциденты на 60%. Повышение эффективности работы операторов на 40%.
