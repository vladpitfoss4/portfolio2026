# ADBISON Website Design

Современный веб-сайт для digital-агентства ADBISON. Проект включает разработку UI/UX дизайна, создание интерактивных прототипов и финальную верстку landing page.

## Задачи проекта

- Разработка визуальной концепции бренда
- Проектирование пользовательских сценариев
- Создание адаптивного дизайна для всех устройств
- Оптимизация конверсии целевых действий

## Технологии

- Figma для дизайна и прототипирования
- Adobe Photoshop для обработки изображений
- Miro для UX-исследований

## Результаты

Увеличение конверсии на 35% после редизайна. Улучшение показателей engagement и снижение bounce rate на 20%.
